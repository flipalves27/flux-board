## Convencoes
- Grid base: 24x24.
- Stroke default: 1.6 (icones lineares) e 2.2 para marcadores de gateway.
- Nomes em formato `bpmn-icon-*`.
- Uso via `<use href="#bpmn-icon-..." />`.

## Variantes de estado
As variantes de estado sao aplicadas por estilo no container (opacidade, stroke scale, glow), sem duplicar simbolos SVG.
