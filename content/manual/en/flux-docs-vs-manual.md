| | **Flux Docs (workspace)** | **This product manual** |
| --- | --- | --- |
| Scope | **Your org** (content, permissions) | The same public help for every signed-in user |
| Storage | Org document storage, tree in app | Versioned help files, no private data |
| AI | May RAG/generate on *your* docs, when enabled | The manual Fluxy reads *this* content, not your files |

This manual is not a substitute for your internal security or contractual record keeping.

## When to use which

- **Workspace docs** — what your team decides to store and link to cards.
- **Manual** — how platform screens, plans, and features are intended to work.
