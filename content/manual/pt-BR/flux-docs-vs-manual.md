---

| | **Flux Docs (workspace)** | **Manual do produto** |
| --- | --- | --- |
| Público | Só a **sua organização** (conteúdo e permissões) | Toda a gente com sessão, mesma base de texto |
| Onde fica | Armazenamento da org, árvore na app | Ficheiros versionados de ajuda, sem dados privados vossos |
| IA | Pode RAG/gerar sobre *os vossos* documentos, se o plano permitir | A Fluxy do manual lê *este* conteúdo, não os vossos ficheiros |

Não use Flux Docs para substituir políticas de segurança ou contratuais — para isso, documentação interna vossa. Use este manual para "como o produto funciona".

## Quando usar cada um

- **Docus do workspace** — processos, decisões, specs que a equipa partilha.
- **Manual** — tutoriais de ecrã, matriz de planos, conceitos oficiais do produto.
